# Split View New Tab

A lightweight Chrome extension that replaces your new tab page with quick access to Google and Grok.

## Features

- Clean, modern dark interface
- - One-click access to Google or Grok
  - - "Open Both" button to launch both sites simultaneously
    - - Tip for using Chrome's native Split View feature
     
      - ## Installation
     
      - 1. **Download the extension**
        2.    - Click the green "Code" button above
              -    - Select "Download ZIP"
                   -    - Extract the ZIP file to a folder on your computer
                    
                        - 2. **Load in Chrome**
                          3.    - Open Chrome and go to `chrome://extensions/`
                                -    - Enable "Developer mode" (toggle in top-right corner)
                                     -    - Click "Load unpacked"
                                          -    - Select the folder containing the extracted files
                                           
                                               - 3. **Done!**
                                                 4.    - Open a new tab to see your Split View launcher
                                                       -    - Click on Google or Grok to open them
                                                            -    - Use "Open Both Side by Side" to launch both at once
                                                             
                                                                 - ## Using Chrome's Native Split View
                                                             
                                                                 - After opening both sites, you can use Chrome's built-in Split View:
                                                                 - 1. Right-click on one of the tabs
                                                                   2. 2. Select "Add tab to new split view"
                                                                      3. 3. Click on the other tab to add it to the split
                                                                        
                                                                         4. ## Note
                                                                        
                                                                         5. Due to security restrictions (X-Frame-Options), Google and Grok cannot be embedded directly in iframes. This extension provides a launcher page instead.
